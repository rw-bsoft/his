7.3.1.6 高血压分组评估

高血压档案保存后，即可打开高血压分组评估页面，如下图：

1)     分组评估页面默认生成一条初次定组的记录。

2)     定转组日期、定转组类型自动默认，不允许修改。定转组类型第一次评估默认为初次定组，第二次及以后新增的默认为不定期转组。

3)     危险分层和血压组别在分组评估保存后，系统根据填写的血压值、危险因素以及系统配置管理->高血压模块参数设置界面中配置的分组规则自动判断并默认，不允许修改。

4)     初次定组的身高、体重、收缩压、舒张压从高血压档案页面中默认过来，可进行修改。

5)     心血管危险因素：包含高血压 （1-3级）、男性＞55 岁或女性＞65 岁、吸烟、糖耐量受损和/或空腹血糖异常、血脂异常、早发心血管病家族史、腹型肥胖或肥胖这些情况。判定为心血管危险因素的条件如下：

l  糖耐量受损在7.8-11.0 mmol/L范围内时自动作为一个心血管危险因素。

l  空腹血糖在6.1-6.9 mmol/L范围内时自动作为一个心血管危险因素。

l  BMI是系统根据身高和体重自动计算的，不可修改。当BMI≥28时判定为腹型肥胖或肥胖，自动作为一个心血管危险因素。

l  腰围填写≥85cm判定为腹型肥胖或肥胖，自动作为一个心血管危险因素。

l  TC(mmol/L)是指总胆固醇，当TC≥5.7mmo1/L为血脂异常，自动作为一个心血管危险因素。

l  LDL-C(mmol/L)是指低密度脂蛋白，当LDL-C＞3.3mmol/L为血脂异常，自动作为一个心血管危险因素。

l  HDL-C(mmol/L)是指高密度脂蛋白，当HDL-C＜1.0mmol/L为血脂异常，自动作为一个心血管危险因素。

6)     靶器官损害：包含左心室肥厚、颈动脉超声IMT＞0.9mm或动脉粥样斑块、颈-股动脉搏波速度 ＞12m/s、踝/臂血压指数<0.9、估算的肾小球滤过率降低或血清肌酐轻度升高、微量白蛋尿或白蛋白/肌酐比异常，判定为靶器官损害的条件如下：

l  Sokolow -Lyons>38mv或 Cornell>2440mm·mms，LVMI：男≥125, 女≥120g/m 2的，都属于左心室肥厚，判定为靶器官损害。

l  男性的血肌酐(μmol/L)含量在115~133μmol/L，或女性的血肌酐含量在107~124μmol/L范围内的，都属于估算的肾小球滤过率降低或血清肌酐轻度升高，判定为靶器官损害。

l  微量尿白蛋白(mg/24h)填写在30~300mg/24h范围内的，都属于微量白蛋尿或白蛋白/肌酐比异常，判定为靶器官损害。

l  男性的白蛋白/肌酐比≥22mg/g或女性的白蛋白/肌酐比≥31mg/g，都属于微量白蛋尿或白蛋白/肌酐比异常，判定为靶器官损害。

7)     伴临床疾患：自动判定伴临床疾患的条件如下：

l  餐后血糖≥11.1mmol/L( 200mg/dL）、糖化血红蛋白（ HbA1c）≥6.5%。

l  血肌酐、蛋白尿、空腹血糖、餐后血糖、糖化血红蛋白5个字段在填写值后，根据值的范围自动勾上伴临床疾患对应的字典选项。

8)     分组评估保存后，系统根据判定出的组别以及系统配置管理->高血压模块参数设置界面中配置的年度开始月份、年度结束月份、生成方式以及分组计划类型自动生成高血压随访计划，第一条随访计划日期为定组的日期。

9)     如果需要重新进行评估，点击“新增”按钮，左边列表上会增加一条不定期转组记录，所有信息默认前一次评估结果，根据需要进行修改，如果未保存就关闭窗口或者去操作其他页面这样系统自动删除该条记录

10)   如果重新评估的管理分组发生变化，随访中未执行的随访计划会根据新的管理分组重新生成。

11)   初次定组保存后可以操作高血压随访页面。
