8.2.1.8 孕产妇产检随访

孕产妇档案保存后，可打开孕产妇产检随访界面，如下图：

1)    页面左侧为随访计划列表，该列表中的所有随访计划是孕产妇档案保存后根据系统配置管理->孕妇模块参数设置界面中配置的生成方式和孕周对应的计划类型以及高危评分自动生成孕产妇产检随访计划，孕产妇随访计划只生成至42孕周。

2)    孕产妇产检随访信息中的检查日期的选择是根据系统配置管理->孕妇模块参数设置界面中配置的提前天数和延后天数进行控制的，检查日期可选择的最小值是当前计划的开始日期（开始日期=当前计划日期-提前天数），检查日期最大值选择的条件是当前计划的结束日期（结束日期=当前计划日期+延后天数）和当前日期的比较，若结束日期＞当前日期，则最大值只能选择当前日期，若结束日期＜当前日期，最大值选择结束日期。

3)    当随访结果选择“已终止妊娠”的，保存随访信息后，该人的孕产妇档案即被终止管理，显示在终止妊娠一览表模块列表中，孕产妇档案及其随访信息都只能查看不允许修改。

4)    胎方位和胎位异常只能≥18孕周时才能填写，＜18孕周不允许填写。

5)    需要对孕妇进行高危评估后才能保存产检随访信息，高危因素的变动会致使孕产妇产检随访计划的重新生成。

6)    当孕产妇产检随访信息填写完整后，点击“确定”按钮，即可保存本次录入的随访信息，且在左侧随访计划列表中自动填充本次随访计划的处理日期。

7)    当孕产妇随访计划生成方式配置为“按照随访结果”生成，保存孕产妇随访后，随访计划重新生成规则如下（已做的随访计划不变）：

l  非高危孕妇：孕妇模块参数设置界面不勾选“所有孕周随访间隔一致”选项的情况下，若建册孕周<=16孕周，随访计划按孕周段计划类型从16孕周的第一天开始生成至42孕周；

l  非高危孕妇：孕妇模块参数设置界面不勾选“所有孕周随访间隔一致”选项的情况下，若建册孕周>16孕周，随访计划按孕周段计划类型从建册孕周的第一天开始生成至第42孕周；

l  非高危孕妇：孕妇模块参数设置界面勾选了“所有孕周随访间隔一致”选项的情况下，孕妇随访计划则按公共计划类型从建册孕周的第一天开始生成至第42孕周。

l  高危孕妇：不管是否勾选了“所有孕周随访间隔一致”选项，随访计划都将根据高危字典规则从建册孕周的第一天开始生成至第42孕周。

8)    当孕产妇产检随访计划生成方式配置为“按照下次预约时间”生成，孕产妇产检随访信息中的下次预约时间即为必填项：

l  下次预约时间的选择是根据系统配置管理->孕妇模块参数设置界面中配置的延后天数进行控制的，只控制下次预约日期最小值的选择，最大值不控制。最小值选择的条件是当前计划的结束日期（结束日期=当前计划日期+延后天数）和当前日期的比较，若结束日期＞当前日期，最小值则选择当前计划结束日期的后一天；若结束日期＜当前日期，最小值则选择当前日期后一天。

l  按下次预约时间随访时，每做完一条孕产妇产检随访计划保存后能且只能生成一条下次预约时间的随访计划。
