11.2.4.3 组织架构维护

1)     在组织架构界面中点击增加部门或增加子部门按钮，可以新建组织架构信息。

2)     部门类型的下拉数据在组织架构类型维护模块中进行维护。

3)     选择部门类型后，右边机构列表中将显示对应组织架构类型的机构或科室信息，已注销的机构或科室不显示。

4)     输入机构代码或科室代码回车，则自动查询已选择部门类型下的符合机构或科室代码条件的机构或科室记录，若存在对应记录，则在列表中显示；若不存在，则弹出机构科室注册界面，后续的机构科室注册操作可参见机构科室注册模块。

5)     当部门类型为“团队(虚拟)”的，组织机构代码不需要填写。

6)     保存后的组织架构的部门类型不允许再修改。

7)     包含子架构的组织架构不允许删除；已经被用户关联了职位的组织架构也不允许删除。
