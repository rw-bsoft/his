11.1.1.5 老年人模块参数设置

在系统菜单栏选择系统管理->系统配置管理->系统配置管理，打开老年人模块参数设置界面，如下图：

1)     年度起止月份：年度开始月份和年度结束月份设置后，建立老年人档案并保存后，系统将根据此处设置的年度开始月份和年度结束月份在本年度列表中生成一年的随访计划。

2)     老年人起始年龄：老年人起始年龄设置后，建立老年人档案时，系统将根据此处设置的年龄值来判断是否已达到建立老年人档案的年龄，若没有达到该年龄，则不允许建立老年人档案。

3)     “所有年龄随访间隔一致”选项与“计划类型”选项是配合一起使用的，当勾选了“所有年龄随访间隔一致”选项时，不管老年人年龄是多少，建立老年人档案后都统一根据此处设置的计划类型来生成老年人随访计划；当不勾选“所有年龄随访间隔一致”选项时，建立老年人档案后将会根据老年人的年龄阶段及其设置的年龄段计划类型来生成老年人随访计划。

4)     老年人计划类型可设置频率为“年”和“月”的计划类型。

5)     生成方式：老年人随访计划生成方式可设置为“按固定时间”和“按下次预约时间”生成。

l  按固定时间：按老年人模块参数设置界面中配置年度起止月份、计划类型/年龄段计划类型来生成一周年的随访计划。

l  按下次预约时间：建立老年人档案保存后则只生成一条当前日期的随访计划，然后每随访一次则按下次预约时间生成一条随访计划。

6)     提前天数和延后天数是为了对随访时间和下次预约时间进行控制。提前天数是指能够提前多少天进行随访，延后天数是指随访可延迟的天数。

7)     年度开始月份和年度结束月份可随时修改；老年人起始年龄在产生老年人档案之前可以修改；计划类型、提前天数、延后天数、生成方式这些参数在产生老年人随访计划之前可修改。
