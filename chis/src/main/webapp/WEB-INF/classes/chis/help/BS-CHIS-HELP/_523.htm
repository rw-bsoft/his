9.5.1.4 新建传染病记录

在传染病管理主界面中，点击“新建”按钮，新建个人健康档案后，即可打开传染病管理页面，如下图：

1)     传染病报告与个档是一对一关系，如果个人档案注销或其他专档死亡、迁出注销后传染病报告卡只读，个人健康档案注销恢复后，传染病报告卡也随之恢复，不单独在注销恢复列表显示。其他专档包括：高血压、糖尿病、精神病、老年人、孕产妇、儿童、血吸虫、残疾人、狂犬病。

2)     因传染病为高敏感数据，EHRVIEW文件夹不包括传染病目录。

3)     新建传染病报告卡时报卡类别默认为初次报告。
