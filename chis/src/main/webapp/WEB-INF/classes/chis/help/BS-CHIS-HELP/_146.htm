7.1.1.5 新建老年人档案

1.      在老年人档案管理主界面中，点击“新建”按钮，即可弹出个人基本信息查询界面，填写个人基本信息并点击“确定”按钮后，系统将根据基本信息中填写的出生日期判别是否达到系统配置管理->老年人模块参数设置界面配置的老年人年龄，若没有达到配置的老年人年龄，即不允许建立老年人档案。若达到了配置的老年人年龄，即可新建个人健康档案以及老年人档案。

2.      当建完个人健康档案后，方可打开老年人档案填写界面，如下图：

1)     档案编号自动从个人健康档案中默认过来，不允许修改。

2)     责任医生自动从个人健康档案中默认过来，当老年人档案保存前，责任医生允许被修改，责任医生可选择为个档管辖机构内的责任医生或团队长。

3)     管辖机构根据责任医生自动带出，不允许修改。

4)     老年人档案保存后，系统将根据系统配置管理->老年人模块参数设置界面中配置的年度开始月份、年度结束月份、生成方式、计划类型以及提前天数和延后天数自动生成老年人随访计划。

5)     老年人档案保存后，档案编号、责任医生和管辖机构不允许再修改。
