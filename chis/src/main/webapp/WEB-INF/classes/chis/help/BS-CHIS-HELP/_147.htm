7.1.1.6 老年人随访

老年人档案保存前，无法打开老年人随访页面，只有当老年人档案保存并建立后方可打开老年人随访页面，老年人随访页面如下图：

1)     页面左侧为随访计划列表，该列表中的所有随访计划是建立老年人档案后根据系统配置管理->老年人模块参数设置界面中配置的年度开始月份、年度结束月份、生成方式、计划类型以及提前天数和延后天数自动生成的老年人随访计划，可在左侧界面下方点击“上一年”、“本年度”或“下一年”切换查看随访计划。

2)     老年人随访信息中的随访日期的选择是根据系统配置管理->老年人模块参数设置界面中配置的提前天数和延后天数进行控制的，随访日期可选择的最小值是当前计划的开始日期（开始日期=当前计划日期-提前天数），随访日期最大值选择的条件是当前计划的结束日期（结束日期=当前计划日期+延后天数）和当前日期的比较，若结束日期＞当前日期，则最大值只能选择当前日期，若结束日期＜当前日期，最大值选择结束日期。

3)     当老年人随访计划生成方式配置为“按照下次预约时间”随访时，老年人随访信息中的下次预约时间即为必填项：

l  下次预约时间的选择是根据系统配置管理->老年人模块参数设置界面中配置的延后天数进行控制的，只控制下次预约日期最小值的选择，最大值不控制。最小值选择的条件是当前计划的结束日期（结束日期=当前计划日期+延后天数）和当前日期的比较，若结束日期＞当前日期，最小值则选择当前计划结束日期的后一天；若结束日期＜当前日期，最小值则选择当前日期后一天。

l  按下次预约时间随访时，每做完一条老年人随访计划保存后能且只能生成一条下次预约时间的随访计划。

4)     老年人随访时转归选择为“终止管理”的，保存随访信息后，该人的老年人档案即会被注销，终止管理的那条随访计划及其后面的随访计划置为“注销”状态，终止管理的那条随访计划之前的随访计划置为“未访”状态。

5)     当老年人随访计划生成方式配置为“按照固定时间随访”时,一个年度的老年人随访计划做完后，根据最后一次随访信息与下一年度年龄自动生成下一年度计划。如果中间有计划过期，且当前已没有计划可做的，系统自动生成自当前日期起一年的计划，此前的计划标记为“未访”。

6)     随访信息页面中，吸烟、饮酒、饮食以及体育锻炼等信息从个人生活习惯中默认过来；既往病史自动从个人既往史中默认过来。

7)     当老年人随访信息填写完整后，点击“确定”按钮，即可保存本次录入的随访信息，且在左侧随访计划列表中自动填充本次随访计划的处理日期。

8)     老年人随访信息保存后，可对其体格检查和中医遍体描述信息进行录入。
