6.4.4.8 妇保批量申请

1.      在批量修改管理医生主界面，点击“妇保批量申请”按钮，弹出批量修改管理医生-妇保批量申请界面，如下图：

2.      点击“新增”按钮，弹出档案查询界面，选择需迁移的档案记录，如下图：

3.      根据各种查询条件查询过滤本中心内中所有的孕妇档案，勾选需要迁移的孕妇档案记录确定后，回到迁移申请界面，如下图：

1)     现妇保医生只能在当前登录的妇保医生所属中心内进行选择，即只能中心内部迁移。

2)     迁移申请保存后，申请单的迁移状态默认为待迁移，待迁移的申请单允许修改。

3)     迁移申请保存后，由本中心的防保科长进行确认。
