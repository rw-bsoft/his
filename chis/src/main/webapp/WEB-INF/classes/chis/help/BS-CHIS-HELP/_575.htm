11.1.1.2 管理方式设置

本模块用于切换网格化管理方式、糖尿病随访、高血压随访、健康体检的录入格式以及网格地址的

显示方式。

在系统菜单栏选择系统管理->系统配置管理->系统配置管理->管理方式设置，打开管理方式设置界面，如下图：

1)     网格化管理方式：参数值包括部分网格化管理和全网格化管理，默认方式为部分网格化管理。该参数在发生与档案有关的业务数据以后就不能作修改。

2)     糖尿病随访录入方式：随访录入方式包括纸质格式和表单格式。

3)     高血压随访录入方式：随访录入方式包括纸质格式和表单格式。

4)     健康体检录入格式：录入方式包括纸质格式和表单格式。

5)     网格地址显示方式：网格地址录入方式包括树状形式和列表形式。选择了一种显示方式后，系统中涉及网格地址的地方都将以统一方式进行显示。

Ø  树状形式的网格地址显示方式如下图：

    双击可展开下级节点，选择至最底层，自动将所选择的地址信息填充至网格地址字段中。

Ø  列表形式的网格地址显示方式如下图：

列表形式的网格地址显示方式，支持拼音码查询，可对当前节点下的子节点进行模糊查询。双击“当前路径”下某个节点，数据可根据所点击节点重新加载该节点下子节点列表数据，选择至最底层，自动将所选择的地址信息填充至网格地址字段中。
