11.2.1.3 机构注册

在机构科室注册界面，点击新建按钮，可打开机构注册界面，如下图：

1)     机构注册保存后才能进行科室注册科室注册操作。

2)     组织机构代码只能由数字、字母、_、- 组成，组织机构代码在系统中是唯一的，不允许重复，机构注册保存后组织机构代码不允许修改。

3)     选择机构类型保存后，该机构即会在组织架构维护模块对应的组织架构类型中显示。

4)     上级部门自动默认为新建机构时选中的机构名称。

5)     机构注册保存后科室注册页面自动启用。

6)     机构上下级关系在机构注册左边树目录中显示。

7)     有子机构的机构不允许注销；有科室和组织架构关联的机构不允许注销。
