7.4.1.5 新建糖尿病档案

在糖尿病档案主界面中，点击“新建”按钮，新建基本信息和个人健康档案后即可新建糖尿病档案。糖尿病档案新建界面如下图：

1)     档案编号自动默认为个人健康档案的编号，不可编辑。

2)     责任医生也自动默认为个人健康档案的责任医生，新建糖尿病档案时可将责任医生修改为其个人建立档案管辖机构内的其他的责任医生或团队长。糖尿病档案保存后，其责任医生不允许再修改。

3)     管辖机构根据责任医生自动带出，不允许修改。

4)     糖尿病家族史和家族史默认继承个人既往史中的家族疾病史，不可以进行修改。

5)     确诊年月默认为当前系统时间，可以修改。

6)     病程=当前系统时间-确诊年月，不可修改。

7)     血糖单位默认为mmol/L，不可编辑。如果填写的血糖值单位为mg/dl，可以点击“血糖单位转换为mmol/L”按钮进行血糖值及单位转换。

8)     糖尿病档案保存后，可维护糖尿病服药情况，服药情况可单独保存，不再与档案绑定保存。

9)     糖尿病档案保存后，系统根据系统配置管理->糖尿病模块参数设置界面中配置的年度开始月份、年度结束月份、生成方式以及分组计划类型自动生成糖尿病随访计划，第一条随访计划日期为糖尿病档案保存的日期。

10)   档案保存后健康检查按钮才能操作，单击健康检查按钮可跳转到健康检查表界面，在健康检查表中检查途径默认为糖尿病。
