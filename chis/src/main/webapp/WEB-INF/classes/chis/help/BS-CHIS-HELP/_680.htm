11.4.3.3 新建肿瘤高危标准

在主界面点击【新建】按钮打开初筛转高危标准维护界面，如下图：

1)        首先维护肿瘤高危标准基本信息，选择年度、高危类别、高危来源和高危因素，填写高危名称。

2)        高危因素包括：问卷症状-有、家族肿瘤史-有、肿瘤疾病史-有；

a)        选择问卷症状-有时：先判断标准中定义的问卷结果与实际问卷结果有一条对上则进入高危，如果问卷结果对不上则判断检查结果，检查结果有一项满足就可进入高危；

b)        选择家族肿瘤史-有时：先判断健康档案中个人既往史--家族史中有“恶性肿瘤”的则进入高危，如果没有则判断检查结果，检查结果有一项满足就可进入高危；

c)        选择肿瘤疾病史--有 时：先判断健康档案中个人既往史--疾病史中有“恶性肿瘤”的则进入高危，如果没有则判断检查结果，检查结果有一项满足就可进入高危；

3)        如果高危因素选择问卷症状-有，保存肿瘤高危确诊维护界面信息后，激活问卷标准列表和检查标准列表标签页；如果高危因素选择家族肿瘤史-有、肿瘤疾病史-有，只激活检查标准列表标签页。

4)        检查标准列表：用以维护初筛转高危的检查标准，如果检查结果与此处维护的项目和结果一致，则进入高危，维护界面如下图：

点击按钮查询检查项目，选定后确认自动回填至检测项目名称中，性质默认为高危，结果分为阴性和阳性2种结果。

问卷标准列表：用以维护进入高危的问卷标准，具体操作可以参考问卷标准维护，此处维护的问卷标准类型为高危，用于是否转高危的判断。
