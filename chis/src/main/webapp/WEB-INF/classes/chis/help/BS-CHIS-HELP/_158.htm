7.1.3.4 新建中医药健康记录

1.      在中医药健康主界面中，点击“新建”按钮，即可弹出个人基本信息查询界面，填写个人基本信息并点击“确定”按钮后，系统将根据基本信息中填写的出生日期判别是否达到系统配置管理->老年人模块参数设置界面配置的老年人年龄，若没有达到配置的老年人年龄，即不允许建立老年人档案。若达到了配置的老年人年龄，即可新建个人健康档案以及老年人档案。

2.      当建完个人健康档案后和老年人档案后，即可激活中医药健康管理标签页

3.      如果已经建立了个人基本信息和个人健康档案的老人，点击新建按钮，在个人基本信息界面用卡号或身份证或姓名+性别+出生日期的方式调出基本信息，点击确定按钮后直接打开中医药健康管理界面，如下图：

1)     界面左侧是历次中医药健康管理记录列表，选中一条记录，界面右侧显示对应记录的详细内容。

2)     点击新建按钮，可新增一条中医药健康管理记录

3)     新增记录时，责任医生或团队长根据页面问题，对老人进行询问并记录对应答案，当33项问题都填写完毕后，点击界面上方【保存】按钮，系统自动根据答案计算出得分和体质类型并显示在中医药保健指导标签页中，点击中医药保健指导标签进行查看，如下图：

        具体的体质判定标准可以参考附录1中的体质判定标准表。

4)     最后医生根据体质判定结果，勾选相应的保健指导，或填写其他相关保健指导，完成所有操作后，点击【确定】按钮保存即可。
