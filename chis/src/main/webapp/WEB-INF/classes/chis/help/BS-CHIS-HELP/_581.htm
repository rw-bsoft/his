11.1.1.8 儿童模块参数设置

在系统菜单栏选择系统管理->系统配置管理->系统配置管理，打开儿童模块参数设置界面，如下图：

1)     建册截至年龄：超过此处设置的建册截至年龄的不能新建儿童基本信息。建册截至年龄可设置至7周岁。

2)     死亡登记截至年龄：超过此处设置的死亡登记截至年龄的不能新建儿童死亡信息。死亡登记截至年龄可设置至7周岁。

3)     首次随访天数：是指儿童从出生到第一次随访的时间间隔。最大可设置至42天。

4)     “所有月龄随访间隔一致”选项与“计划类型”选项是配合一起使用的，当勾选了“所有月龄随访间隔一致”选项时，不管儿童月龄是多少，建立儿童档案后都统一根据此处设置的计划类型来生成儿童询问计划和体格检查计划；当不勾选“所有年龄随访间隔一致”选项时，建立儿童档案后将会根据儿童的月龄段及其设置的月龄段计划类型来生成儿童询问计划和体格检查计划。

5)     儿童计划类型可设置频率为“年”和“月”的计划类型。

6)     提前天数和延后天数是为了对随访时间和下次预约时间进行控制。提前天数是指能够提前多少天进行随访，延后天数是指随访可延迟的天数。

7)     建册截至年龄在产生儿童档案之前可修改；死亡登记截至年龄在产生儿童死亡记录之前可修改；首次随访天数可随时修改；提前天数、延后天数、计划类型这些参数在产生儿童询问和体检计划之前可修改。
