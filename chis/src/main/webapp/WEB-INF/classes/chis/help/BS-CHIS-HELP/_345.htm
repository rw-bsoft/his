7.8.1.2 管理权限

心血管监测列表模块，只有责任医生、团队长、中心主任、网络管理员、防保科长、市CDC、区CDC、市区卫生局这些角色的用户可以访问。其中：

1)     责任医生角色登录：心血管监测列表数据按责任医生过滤，只显示责任医生为本人的心血管监测记录；个人健康档案的责任医生可以新建、修改和删除心血管监测信息。

2)     团队长角色登录：心血管监测列表数据按管辖机构(团队)过滤；个人健康档案的责任医生可以新建、修改和删除心血管监测信息。

3)     中心主任、网络管理员、防保科长角色登录：心血管监测列表数据按管辖机构(中心)过滤；对心血管监测信息只读。

4)     市CDC角色登录：心血管监测列表数据按管辖机构(市)过滤；对心血管监测信息只读。

5)     区CDC角色登录：心血管监测列表数据按管辖机构(区)过滤；对心血管监测信息只读。

6)     市区卫生局角色登录：心血管监测列表数据按管辖机构(市/区)过滤；对心血管监测信息只读。
