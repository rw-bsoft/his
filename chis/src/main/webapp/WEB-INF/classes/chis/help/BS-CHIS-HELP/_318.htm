7.7.1.6 训练评估

当训练登记保存后，即可打开训练评估页面，如下图：

1)     初次评估保存后，不可操作总结页面，总结页面是在一次评估周期（一次评估周期是指完成评估->训练计划->训练记录这样一个流程）结束后重新进行不定期评估时才填写的期中总结。

2)     初次训练评估保存后，可操作训练计划与记录页面；

3)     不定期评估3个月必须评估1次，超过3个月未进行评估的，系统自动在我的首页->任务列表中给出训练评估的任务提醒。

4)     不定期训练评估的最小间隔为1个月，只能间隔1个月及以上时间才能评估一次。

5)     不定期评估保存后，系统会提示是否需要重新调整训练计划。

6)     不定期评估保存后，可填写期中总结信息。
