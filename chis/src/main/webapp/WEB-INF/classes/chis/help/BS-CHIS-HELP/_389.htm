8.1.2.2 管理权限

儿童档案管理模块，责任医生、团队长、儿保医生、中心主任、网络管理员、防保科长、市妇幼、区妇幼和市区卫生局这些角色的用户可以访问。其中：

1)     责任医生角色用户登录：儿童档案列表数据按责任医生过滤，只显示责任医生为本人的儿童档案记录；对儿童档案仅只读查看。

2)     团队长角色用户登录：儿童档案列表数据按管辖机构（中心）过滤，只显示中心内的儿童档案记录；对儿童档案仅只读查看。

3)     儿保医生角色用户登录：儿童档案列表数据按管辖机构或建档人/录入人为当前登录者过滤；可以新建和修改全市的儿童档案。

4)     中心主任、网络管理员和防保科长角色用户登录：儿童档案列表数据按管辖机构（中心）过滤；对儿童档案仅只读查看。

5)     市妇幼角色用户登录：儿童档案列表数据按管辖机构（市）过滤；对儿童档案仅只读查看。

6)     区妇幼角色用户登录：儿童档案列表数据按管辖机构（区）过滤；对儿童档案仅只读查看。

7)     市区卫生局角色用户登录：儿童档案列表数据按管辖机构（市/区）过滤；对儿童档案仅只读查看。
