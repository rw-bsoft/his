11.4.2.3 新建筛查检查项目

点击主界面的新建按钮，打开筛查检查项目新建界面，如下图：

1)        必填项：项目类别、费用归并、项目名称、是否启用

2)        项目类别：对应肿瘤的高危类别，只有对应的肿瘤疑似病人才可以做该筛查检查项目；

3)        费用归并：选择该筛查检查项目的费用归并类别。

4)        项目名称：在费用归并字段选择一个类别后，项目名称方可录入，否则置灰，无法选择。点击按钮，打开检查项目查询界面，如下图：

检查项目查询界面根据费用归并类别自动过滤对应类别的检查项目，并显示在界面下方列表中。在列表中选择一个项目打钩，点击【确定】按钮后，所选项目会自动填入到项目名称中。

5)        是否启用：系统默认值为是，如果选择否，则维护的该项目无法使用。
