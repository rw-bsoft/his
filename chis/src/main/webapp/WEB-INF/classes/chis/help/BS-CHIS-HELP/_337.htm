7.7.3.5 新建训练登记

在智力残疾登记主界面中，点击“新建”按钮，对于14周岁以内的人新建基本信息和个人健康档案后即可新建智力残疾登记。智力残疾登记界面如下图：

1)     训练登记保存前，不允许操作训练评估和训练计划与记录页面。

2)     训练登记状态为正常且未结案时，不允许再次新建训练登记。

3)     训练登记时，责任医生可以选择当前责任医生所属机构中心内的责任医生或团队长。

4)     新建训练登记保存后，系统自动将智力残疾情况同步到个人既往史信息中。

5)     训练登记保存后，系统自动提示是否进行训练评估。训练评估页面可进行操作。
