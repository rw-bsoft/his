7.6.3.2 管理权限

T疑似人群管理模块，只有责任医生、团队长、中心主任、网络管理员、防保科长、防保员、市CDC、区CDC和市区卫生局角色用户可以访问。其中：

1)     责任医生角色用户登录：T疑似人群管理列表数据按责任医生过滤；个档管辖机构内的责任医生可以新建、过滤、复核T疑似人群；录入人可以修改和删除T疑似人群记录；

2)     团队长角色用户登录：T疑似人群管理列表数据按管辖机构(团队)过滤；个档管辖机构内的团队长可以新建、修改、删除、过滤复核T疑似人群档案；

3)     防保员用户登录：T疑似人群管理列表数据按管辖机构(中心)过滤；个档管辖机构内的防保员可以新建、修改、删除、过滤复核T疑似人群档案。

4)     中心主任、网络管理员、防保科长角色用户登录：T疑似人群管理列表数据按管辖机构(中心)过滤；

5)     市CDC角色用户登录：T疑似人群管理列表数据按管辖机构(市)过滤；

6)     区CDC角色用户登录：T疑似人群管理列表数据按管辖机构(区)过滤；

7)     市区卫生局角色角色用户登录：T疑似人群管理列表数据按管辖机构(市/区)过滤；
