7.5.1.6 精神病首次随访

当填写精神病档案信息后点击“确定”按钮，系统将自动转到精神病首次随访界面，如下图：

1)     本次随访分类由危险性、目前症状、社会功能情况等自动判断，不允许修改：

l  不稳定：危险性为3～5级，自知力缺乏，药物不良反应。

l  基本稳定：若危险性为1～2级，自知力不全、社交能力较差。

l  稳定：危险性为0级，且自知力完全，社交能力处于一般或良好，无药物不良反应。

2)     服药依从性选择“规律”或“间断”时，可填写服药情况。

3)     当精神病首次随访信息填写完整后，返回精神病档案页面保存档案后，系统将根据首次随访中判定的随访分类和系统配置管理->精神病模块参数设置界面中的年度开始月份、年度结束月份、随访分类计划类型自动生成一条精神病随访计划，计划日期与首次随访日期间隔时间根据随访分类判定。
