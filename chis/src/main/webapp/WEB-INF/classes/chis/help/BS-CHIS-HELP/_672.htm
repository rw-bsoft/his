11.4.1.3 新建问卷标准

点击主界面的新建按钮，打开问卷标准新建界面，如下图：

1)        新建问卷标准界面上方为当前所维护的标准的基本信息，基本信息字段包括：管理年度、高危类别、标准类别、模板名称、标准判别，所有字段均为必填项；

2)        管理年度：新建时默认显示当前年份，通过下拉框可选择以往年度以及下一年度；

3)        高危类别：维护标准所适应的高危类别，高危类别包括：大肠、胃、肝、肺、宫颈；

4)        标准类别：此处新建的问卷标准，标准类别默认为初筛，不可修改；

5)        模板名称：在模板名称文本框右侧点击按钮，打开问卷模板查询界面，如下图：

       问卷模板查询界面打开时，维护模版的值自动默认为标准维护界面上的高危类别的值，并根据当前管辖机构自动查询出符合条件的模板。

       在问卷模板查询界面下方的模板列表中，选择一个模板打上勾，点击【确定】按钮后自动返回到问卷标准维护界面，并将所选问卷模板的字段名称、字段值信息显示在下方界面中。

6)        标准判别：选择“所有”选项时，做问卷时，只要做了问卷就纳入相应的标准类别。选可疑时，将问卷的问题结果与标准的问题字段/值有某个对上后即纳入相应的标准类别。

7)        问题字段及字段值的维护：根据第5点选择问卷后，所选问卷的问题、问题答案值会显示在界面下方，左侧栏是问题列表，点击一个问题，中间栏显示所选问题包含的答案值，双击其中一个答案值，系统自动将所选答案作为该问题的问卷标准值显示在右侧栏中。当居民在做该问卷时，有一个答案符合该问卷标准值的，就会被纳入初筛人员类别中。以此类推可以给每个问题维护标准值。如下图：
