11.6.2.4 家庭签约模板维护

在模板维护主界面，点击【新建】按钮打开模板维护界面，如下图

1)     红色显示的字段为必填项，维护模板选项包括家庭签约服务记录、指标数据维护和肿瘤模板。如果是新建家庭签约服务记录模板则选择“家庭签约服务记录”选项；基础数据维护的，类型选择指标数据维护。

2)     家庭模板类型包括服务记录模板和健康问卷模板，服务记录模板要选择选择服务记录，新建健康问卷的，类型选择健康问卷。

3)     模板名称不可重复，管辖机构根据登录用户默认。

4)     已经被使用过的模板不允许修改和删除。

5)     系统管理员只能查看、不可修改中心管理员维护的模板。
