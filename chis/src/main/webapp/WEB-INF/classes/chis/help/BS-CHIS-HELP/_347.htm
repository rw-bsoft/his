7.8.1.4 心血管评测表单

年龄≥35周岁且建立过个人健康档案的人，可以新建心血管监测信息，心血管监测默认打开心血管评测表单，如下图：

1)     是否吸烟：必输，默认为个人生活习惯的是否吸烟情况，可以修改。

2)     身高、体重：必输，默认读取高血压、糖尿病随访中最新的身高体重数据，可以修改。

3)     腰臀比：计算方式为腰围/臀围，不能修改。

4)     是否糖尿病：必输，根据是否建有糖尿病档案自动填写，可以修改。

5)     空腹血糖、餐后血糖：必输，默认读取糖尿病随访中最新的血糖数据，可以修改。

6)     是否高血压：必输，根据是否建有高血压档案自动填写，可以修改。

7)     收缩压、舒张压：必输，默认读取高血压随访中最新的血压数据，可以修改。

8)     心血管评测表单信息保存后，评测结果表、知识检验与危险预测图页签可用。
