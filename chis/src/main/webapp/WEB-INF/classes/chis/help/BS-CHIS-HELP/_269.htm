7.6.1.5 添加听课人员

在主界面点击【听课人员登记】按钮打开听课人员登记界面，如下图：

1)        听课人员登记界面分为三部分，左边为当前课程的基本信息，只做查看，不能修改；一般情况是隐藏的，点击界面上的按钮，可以展开进行查看。中间显示当前课程的听课人员，可进行新增、查看、删除。右边部分显示当前选定的听课人员所做的问卷记录。

2)        在听课人员登记界面，点击表单中间的【新建】按钮添加听课人员，听课人员必须已经建立健康档案，否则不能添加。听课人员添加后可以查看和删除。

3)        在听课人员列表选中人员后，在听课人员登记界面最右侧点击【新增问卷】按钮，可以给所选人员做健康问卷，如下图：

4)        在肿瘤健康问卷选择模板界面选择需要做的问卷，双击问卷或点击【选择】按钮打开问卷界面，如下图：

问卷上半部分为人员和问卷的基本信息，红色字体标注的字段为必填项，每次做问卷必须填写，其中姓名、性别、出生年月自动从个人基本信息中默认过来，不能修改。类别和来源字段根据当前问卷类别和问卷来源进行默认，不能修改。问卷下半部分为问卷具体问题和答案选项。

问卷完成后点击【确定】按钮即可保存，并自动关闭当前问卷界面，返回到听课人员登记界面，在该界面中可以对已经完成的问卷进行查看和修改。

5)        听课人员完成问卷保存后，系统将根据所维护的问卷初筛标准判断做问卷的人员是否满足进入初筛人群，如果满足，则自动将此人列入初筛人群，如果不满足，则不做任何操作，只保留问卷信息。需注意的是：每次修改问卷时都会重新判断是否满足初筛标准，所以每次修改并保存问卷，如果满足初筛标准，都会重新一条初筛记录。
