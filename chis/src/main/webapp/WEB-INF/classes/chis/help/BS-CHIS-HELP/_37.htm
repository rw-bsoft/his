6.2.1.5 家庭档案(全网格化管理)

全网格化管理模式进行家庭档案管理，需要在系统配置管理-管理方式设置模块中，将网格管理方

式选择为“全网格化管理”，修改配置后，需重新登录系统，配置才能生效。

在家庭档案管理主界面中点击“新建”按钮，即可开始新建家庭档案，如下图：

1)     基本信息表单数据未保存时，成员列表和问题列表tab页不可用。

2)     基本信息表单中，家庭编号、户主姓名、责任医生、网格地址和管辖机构为必填项。

3)     家庭编号在家庭档案保存后根据所选的网格地址编号自动生成，不需要手动填写。

4)     户主姓名可自主填写；当选择网格地址后，户主姓名下拉框中可过滤出当前选择的网格地址上已建有个人健康档案记录的个人姓名，此时可选择其中之一作为家庭的户主。

5)     责任医生默认与所选网格地址上维护的责任医生一致，当所选网格地址上没有维护责任医生时，该页面的责任医生可以选择角色是责任医生和团队长角色的医生。

6)     管辖机构根据所选的责任医生自动默认，不需要手动填写。

7)     基本信息表单数据保存后，成员列表和问题列表tab页启用，档案编号、责任医生、网格地址和管辖机构不允许再修改。

8)     通过成员列表tab页，可将已建有个人健康档案的个人自动匹配到本家庭中，对本家庭成员进行维护。

9)     通过问题列表tab页，可维护本家庭的主要问题。
