6.4.2.5 防保科长确认

1.      防保科长登录系统，如果存在需要确认的迁移申请时，系统自动在我的首页->任务列表中给出确认儿童户籍地址迁移的任务提醒。

2.      防保科长在儿童户籍地址迁移列表选择待确认的儿童档案迁移申请记录，点击“确认”按钮，打开确认窗口，如下图：

1)     防保科长确认时，可修改现儿保医生，填写确认人意见信息。

2)     确认时点击退回，即不同意该迁移申请，退回的申请单申请迁移的档案的归属地、户籍地址和儿保医生不变。

3)     退回后申请单迁移状态更新为已退回，退回的申请单不允许修改和删除。

4)     确认时点击执行后，即表示同意该申请。儿童档案迁移申请执行后，被申请的儿童档案和体弱儿童档案的归属地、户籍地址、儿保医生和管辖机构被修改。

5)     已注销的档案也一同被迁移。

6)     执行后申请单迁移状态更新为已确认，已确认的申请单不允许被修改和删除。
