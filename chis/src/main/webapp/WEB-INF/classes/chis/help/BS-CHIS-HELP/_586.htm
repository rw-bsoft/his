11.1.1.13 肿瘤高危人群随访参数设置

在系统菜单栏选择系统管理->系统配置管理->系统配置管理，打开肿瘤高危人群模块参数设置界面，如下图：

1)     管理年度：设置肿瘤高危随访计划生成时年度开始、结束月份，例如设置开始月份是9月份，结束月份是8月份，那么离休干部随访计划生成时1至8月份的计划显示在本年度，9月份以后开始下一年的随访计划。

2)     计划类型：随访的频率。

3)     提前天数和延后天数是为了对随访时间和下次预约时间进行控制。提前天数是指能够提前多少天进行随访，延后天数是指随访可延迟的天数。

4)     随访分组：分为常规组和高危组。

5)     高危类别：可以给不同肿瘤类别制定不同频率的随访计划。

6)     年度开始月份和年度结束月份可随时修改；计划类型、提前天数、延后天数这些参数在产生肿瘤高危随访计划之前可修改。
