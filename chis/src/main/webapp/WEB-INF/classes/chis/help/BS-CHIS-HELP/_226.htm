7.4.2.2 管理权限

糖尿病随访记录模块，只有责任医生、团队长、中心主任、网络管理员、防保科长、市CDC、区CDC、市区卫生局这些角色的用户可以访问。其中：

1)     责任医生用户登录：糖尿病随访记录列表数据按糖尿病档案的责任医生过滤，只显示责任医生为本人的糖尿病随访记录；糖尿病档案管辖机构（中心）内的责任医生都可以修改糖尿病随访记录。

2)     团队长角色用户登录：糖尿病随访记录列表数据按管辖机构(团队)过滤；糖尿病档案管辖机构（中心）内的团队长都可以修改糖尿病随访记录。

3)     中心主任、网络管理员、防保科长角色用户登录：糖尿病随访记录列表数据按管辖机构(中心)过滤；对糖尿病随访记录只读。

4)     市CDC用户登录：糖尿病随访记录列表数据按管辖机构(市)过滤；对糖尿病随访记录只读。

5)     区CDC用户登录：糖尿病随访记录列表数据按管辖机构(区)过滤；对糖尿病随访记录只读。

6)     市区卫生局角色用户登录：糖尿病随访记录列表数据按管辖机构(市/区)过滤；对糖尿病随访记录只读。
