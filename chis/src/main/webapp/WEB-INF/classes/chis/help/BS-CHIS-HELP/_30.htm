6.1.3.3 主界面

在系统菜单栏选择工作计划->儿保工作列表，打开儿保工作列表主界面，如下图：

1)     列表视图：可以通过各种查询字段和日期查询儿童询问、儿童体格检查、体弱儿随访计划任务，处理每条计划任务。也可以按照“所有任务”、“正常任务”、“紧急任务”、“过期任务”、“可执行任务”和“未到期任务”这些任务类型来查询和过滤数据。

l  所有任务：条件范围内的所有任务。

l  正常任务：条件范围内非过期的任务。

l  紧急任务：剩余天数≤3天的任务。

l  过期任务：已经超过最晚执行日期的任务。

l  可执行任务：在最早执行日期和最晚执行日期期间的任务。

l  未到期任务：未到最早执行日期的任务。

2)     随访：在儿保工作列表中选择一条儿童询问/儿童体格检查/体弱儿随访计划任务，点击“随访”按钮，可打开儿童儿童询问/儿童体格检查/体弱儿随访记录页面，之后的操作请参考儿童询问记录、儿童体格检查和体弱儿随访模块说明。
