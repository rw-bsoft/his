11.2.3.3 用户注册

在用户注册界面中点击新建按钮，可打开用户新建界面，如下图：

1)     点击证件号码的选择按钮，可打开人员选择窗口，窗口中正确显示系统中所有已注册的人员记录；人员信息选择窗口中也可对人员进行新建和修改。

2)     选择人员后，人员编码、人员姓名和证件号码自动填充到用户注册界面中。

3)     已经添加为用户的人员不允许重复添加。

4)     用户状态可设置为正常或禁用，禁用的用户无法登录系统。

5)     用户信息保存后，可以操作用户角色关系。

6)     新增用户角色关系时，职位可以选择所属域下不同的机构里的不同角色，支持用户在不同的机构中使用，职位由角色管理、组织架构类型以及组织架构维护拼装而成。

7)     一个用户必须有一个用户角色关系才允许保存，用户角色关系可以维护多条记录。

8)     同一个用户同一个角色可隶属不同的管辖机构。

9)     同一个用户在同一个团队中不能同时拥有责任医生和团队长的用户属性。
