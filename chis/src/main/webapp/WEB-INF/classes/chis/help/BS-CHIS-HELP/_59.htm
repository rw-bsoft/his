6.2.4.4 档案注销恢复随访计划的调整

一、              老年人、糖尿病、精神病档案注销恢复随访计划的调整

1.      通过档案注销->恢复(恢复status=9的随访计划，恢复日期与最后一条计划的enddate比较)

1)     按固定时间/随访结果配置的：

l  恢复日期＞enddate，根据配置生成一周年的随访计划，生成的首条随访计划的随访日期为恢复日期。

l  恢复日期≤enddate，不重新调整随访计划。

2)     按下次预约时间配置的：

l  恢复日期＞enddate，生成一条恢复日期的随访计划，之前的计划置为“未访”。

l  恢复日期≤enddate，不重新调整随访计划。

2.      通过随访终止管理注销->恢复(恢复status=9的随访计划，恢复日期与最后一条计划的enddate比较)

1)     按固定时间/随访结果配置的：

l  当恢复日期＞enddate，终止管理的计划置为“未访”，根据配置生成一周年的随访计划，生成的首条随访计划的随访日期为恢复日期。

l  恢复日期≤enddate，终止管理的随访记录删除并恢复到未执行状态，状态为“应访”。

2)     按下次预约时间配置的：（精神病档案注销恢复随访计划的调整仅参考此说明）

l  恢复日期＞enddate，终止管理的计划置为“未访”，生成一条恢复日期的随访计划。

l  恢复日期≤enddate，终止管理的随访记录删除并恢复到未执行状态，状态为“应访”。

二、              体弱儿档案注销恢复随访计划的调整：

恢复status=9的计划，恢复日期与最后一条计划的enddate比较。

l  恢复日期＞enddate，生成一条恢复日期的随访计划。

l  恢复日期≤enddate，不重新调整随访计划。
