6.2.6.4 贫困人群随访

在个人健康档案管理页面中，点击“新建”按钮，新建个人基本信息和个人健康档案，在个人健康档案页面中，选择“经济来源”包含“社会救济”，并保存个人健康档案，如下图所示：

个人健康档案页面，“经济来源”选项的选择则即决定了是否可对该人进行贫困人群随访，“经济来源”选项选择时，只要包含了“社会救济”，即可对其进行贫困人群随访。否则不能进行贫困人群随访。

在EHRView->基本档案中点击“贫困人群随访”，如下图所示：

点击“贫困人群随访”后即打开了贫困人群随访页面，如下图：

1)     随访日期默认为当前日期，不允许选择未来日期，贫困人群随访信息保存后，随访日期不允许修改。

2)     随访医生默认为个人健康档案的责任医生，允许选择个人健康档案管辖机构内的所有责任医生或团队长角色。

3)     收入来源默认为个人健康档案页面中选择的经济来源。修改贫困人群随访页面的收入来源后，个人健康档案页面的经济来源也同步更新，当收入来源选择时不包含“社会救济”，贫困人群随访信息保存后，则不允许再进行贫困人群随访。

4)     新建贫困人群随访保存后，只允许录入者可修改，其他只允许查看。

5)     新建保存的随访记录在系统菜单栏选择健康管理->健康档案->贫困人群随访模块列表中统一管理，如下图：

6)     个人健康档案注销后不允许再新建或修改贫困人群随访记录，只允许查看。
