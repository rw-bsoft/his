6.2.8.5 健康检查表（纸质录入模式）

表单录入格式录入健康检查表，需要在系统配置管理-管理方式设置模块中，将健康检查表录入格

式选择为“纸质格式”，修改配置后，需重新登录系统，配置才能生效。

新建个人基本信息和个人健康档案后即可打开健康检查表（纸质录入）页面，如下图：

 

1)        纸质录入模式的健康检查表中的住院治疗情况、主要用药情况、非免疫规划预防接种史数据保存后会在住院治疗用药情况、非免疫规划预防接种标签页列表中显示；同理，在住院治疗用药情况、非免疫规划预防接种标签页中录入的数据也会在健康检查表的住院治疗情况、主要用药情况、非免疫规划预防接种史字段中显示。但由于健康检查表中的数据显示数量有限，因此系统只显示录入时间考前的几条数据。
