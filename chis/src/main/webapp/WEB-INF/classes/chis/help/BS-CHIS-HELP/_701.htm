11.6.3.3 社区有效性认证流程

流程说明：

1)     登录社区系统后，在系统管理->移动后台维护菜单下打开服务有效性认证模块，如下图：

2)     自动匹配：服务有效性认证列表中，可以对所有的GPS地址定位正常的未匹配状态的服务记录进行自动匹配，将个人健康档案的网格地址与GPS定位地址进行范围匹配，如果在范围内，则更改有效性标识为“有效”，否则为“无效”。

3)     手工质检：对GPS地址标识为0（未知地址）或9（无法定位）的记录进行手工标识，选中某一条记录点击手工质检按钮，可以对该条记录直接做有效或无效处理。如下图：
