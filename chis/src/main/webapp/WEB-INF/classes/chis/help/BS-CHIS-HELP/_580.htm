11.1.1.7 糖尿病模块参数设置

在系统菜单栏选择系统管理->系统配置管理->系统配置管理，打开糖尿病模块参数设置主界面，如下图：

1)     管理年度：设置糖尿病随访计划生成时年度开始、结束月份，例如设置年度开始月份是9月份，年度结束月份是8月份，那么糖尿病随访计划生成时1至8月份的计划显示在本年度，9月份以后的显示在下一年度中。

2)     生成方式：糖尿病随访计划的生成方式可设置为“按照随访结果”和“按下次预约时间”生成。

l  按照随访结果（不满意）：如果随访结果不满意（不满意表现为：空腹血糖≥7.0，或有药物不良反应），则重新生成随访计划，生成原计划的当前随访日期和下一次随访计划日期之间的2周1次的随访计划，其它原有的计划被删除。随访结果如果连续两次血糖控制不满意、连续两次药物不良反应、第一次异常第二次有新的并发症出现或原有并发症加重系统提示转诊，计划仍为2周1次。

l  按照随访结果（满意）：如果随访结果满意，则按原计划生成一周年的随访计划。

l  按下次预约时间：建立糖尿病档案保存后则只生成一条当前日期的随访计划，然后每随访一次则按下次预约时间生成一条随访计划。

3)     提前天数和延后天数是为了对随访时间和下次预约时间进行控制。提前天数是指能够提前多少天进行随访，延后天数是指随访可延迟的天数。

4)     计划类型：设置不同严重级别的糖尿病病人对应的随访计划类型，目前系统没有实现糖尿病级别的划分评定，所有的糖尿病档案生成随访计划时按照糖尿病一组的计划类型生成。

5)     糖尿病计划类型可设置频率为“月”和“周”的计划类型。

6)     年度开始月份和年度结束月份可随时修改；生成方式、提前天数、延后天数、计划类型这些参数在产生糖尿病随访计划之前可修改。
