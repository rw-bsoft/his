7.7.3.9 智力残疾注销

在智力残疾登记列表中选择需注销的档案记录，点击“注销”按钮，即可开始注销操作，如下图：

1）档案注销原因分为死亡、迁出、失访、拒绝、作废和其他。

2）档案注销原因选择“死亡”或“迁出”时，确定注销后将会注销该人的个人健康档案、智力残

疾档案以及其他所有子档案及随访计划；当选择档案注销原因为“失访”、“拒绝”或“其他”时，确定注销后则只会注销该人的所有智力残疾档案，其他档案及其随访计划不会被注销；当选择档案注销原因为“作废”时，确定注销后则只会注销当前选择的那一份智力残疾档案，该人其他的智力残疾档案以及其他子档案都不会被注销。

3）注销后的智力残疾档案显示在已注销列表中，注销后的档案只能查看，不能修改。

4）注销原因为死亡、迁出、失访、拒绝和其他的智力残疾档案注销后可在档案注销恢复管理模块

进行恢复，作废的智力残疾档案不可恢复。
