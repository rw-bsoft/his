6.2.3.2 管理权限

个人健康档案模块，只有责任医生、团队长、儿保医生、妇保医生、中心主任、网络管理员、防保科长、防保员、区域灭螺组和市区卫生局这些角色的用户可以访问。其中：

1）  责任医生角色用户登录：个人健康档案列表数据按责任医生过滤，只显示责任医生为本人的个

人健康档案记录；允许跨区建立个人健康档案；只允许修改和注销责任医生为本人的个人健康档案记录。

2）  团队长角色用户登录：个人健康档案列表数据按管辖机构(团队)过滤；允许跨区建立个人健康

3）  妇保医生、儿保医生、防保员、区域灭螺组角色用户登录：个人健康档案列表数据按管辖机构(中心)过滤；允许跨区建立个人健康档案；无修改和注销个人健康档案的权限。

4）  中心主任、网络管理员、防保科长角色用户登录：个人健康档案列表数据按管辖机构(中心)过滤；对个人健康档案只读。

5）  市区卫生局角色用户登录：个人健康档案列表数据按管辖机构(市/区)过滤；对个人健康档案只读。
