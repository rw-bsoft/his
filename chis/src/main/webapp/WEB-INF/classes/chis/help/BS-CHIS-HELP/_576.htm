11.1.1.3 外部及接口设置

本模块用于社区系统连接到的MPI2.2（病人信息主索引平台）和PHIS2.4.(基层医疗信息系统)接口的相关参数的设置，配置该模块参数后需重启应用程序服务方可生效。

在系统菜单栏选择系统管理->系统配置管理->系统配置管理，打开外部服务接口设置界面，如下图：

1)     MPI/基层医疗系统与社区连接同一个zookeeper，改变设置后，重启服务后设置生效。

2)     是否调用平台PIX服务设置为否的情况下保存设置后，只在社区系统中进行个人信息的查询、新建和更新操作。

3)     是否调用平台PIX服务设置为是的情况下，保存设置重启服务后，在个人基本信息界面中通过卡号、身份证或姓名+性别+出生日期查询个人基本信息时：

l  先在社区库中进行查询，如果存在记录，变成更新操作，确定后更新到社区数据库。

l  如果社区库中无法查询到相关信息，向平台PIX提交该查询，如果平台查询出数据，则将该数据作为默认数据填充到个人基本信息界面上，再由用户对数据进行更新，更新后的数据会同时保存到社区和平台库中。

l  如果社区库中无法查询到相关信息，向平台PIX提交该查询，如果平台查询不到数据，用户将信息填写完整进行全新注册，新注册的信息也会同时保存到社区和平台数据库中。

4)     是否启用基层医疗，设置为是的情况下，保存设置重启服务后，即可在的医生工作站中调用社区卫生服务信息系统的相关业务操作。
