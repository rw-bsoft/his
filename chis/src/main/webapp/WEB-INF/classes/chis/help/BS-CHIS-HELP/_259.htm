7.5.1.7 精神病随访管理

精神病档案保存后，可打开精神病随访管理标签页，随访管理界面如下图：

1)    页面左侧为随访计划列表，该列表中的随访计划是糖尿病档案保存或精神病随访后根据系统配置管理->精神病模块参数设置界面中配置的年度开始月份、年度结束月份、随访分类计划类型、提前天数和延后天数自动生成的精神病随访计划，可在左侧界面下方点击“上一年”、“本年度”或“下一年”切换查看随访计划。

2)    精神病随访信息中的随访日期的选择是根据系统配置管理->精神病模块参数设置界面中配置的提前天数和延后天数进行控制的，随访日期可选择的最小值是当前计划的开始日期（开始日期=当前计划日期-提前天数），随访日期最大值选择的条件是当前计划的结束日期（结束日期=当前计划日期+延后天数）和当前日期的比较，若结束日期＞当前日期，则最大值只能选择当前日期，若结束日期＜当前日期，最大值选择结束日期。

3)    精神病随访时转归选择为“终止管理”的，保存随访信息后，该人的精神病档案以及其随访计划、精神病记录纸和精神病年度评估即同时被注销，不允许再修改。

4)    服药依从性选择“不服药”时服药情况标签页不可用，选择“规律”或“间断”时服药情况标签页可用。

5)    保存一条精神病随访后，系统即根据随访信息中判定的随访分类和系统配置管理->精神病模块参数设置界面中的年度开始月份、年度结束月份、随访分类计划类型自动生成一条精神病随访计划。
