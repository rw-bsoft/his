6.2.4.3 档案恢复

在档案注销恢复管理主界面中，选择一条档案记录，点击“恢复”按钮，即可弹出档案列表界面，如下图：

1)     档案列表中显示所有类型的档案及其档案状态，包括个人健康档案、老年人档案、高血压档案、糖尿病档案、孕妇档案、儿童档案、体弱儿童档案、精神病档案、血吸虫病档案、狂犬病档案和残疾人档案。

2)     档案状态主要分为两种：“正常”和“已注销”。“正常”状态的档案不能恢复，选中“正常”状态的档案记录时恢复按钮不可用；“已注销”的档案可以恢复，选中“已注销”状态的档案记录时恢复按钮可用。

3)     当个人健康档案为“已注销”状态时，所有子档都不能恢复，必须恢复个人健康档案的同时恢复所有子档，或者先恢复个人健康档案后，再分别对每个子档进行恢复。
