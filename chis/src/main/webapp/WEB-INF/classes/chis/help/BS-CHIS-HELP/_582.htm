11.1.1.9 体弱儿模块参数设置

在系统菜单栏选择系统管理->系统配置管理->系统配置管理，打开体弱儿模块参数设置界面，如下图：

1)     计划类型：没有勾上例外情况时按照此设置生成体弱儿计划。

2)     例外情况：

l  勾上时，按照体弱儿档案->体弱儿分类中对应的计划类型生成随访计划。

l  不勾时，按照上面设置的计划类型生成计划。

3)     当体弱儿档案同时满足多种例外情况时，按照例外情况中访问频次最高的生成随访计划。

4)     体弱儿计划类型可设置频率为“月”和“周”的计划类型。

5)     提前天数和延后天数是为了对随访时间和下次预约时间进行控制。提前天数是指能够提前多少天进行随访，延后天数是指随访可延迟的天数。

6)     所有的参数在产生体弱儿随访计划之前可修改，产生体弱儿随访计划之后不可修改。
