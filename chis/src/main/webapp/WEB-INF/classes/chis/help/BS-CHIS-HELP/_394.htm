8.1.2.7 儿童询问记录

儿童健康档案信息保存后，儿童询问记录页面可用，打开儿童询问记录标签页，如下图：

1)    页面左侧为询问计划列表，该列表中的所有询问计划是儿童档案保存后根据系统配置管理->儿童模块参数设置界面中配置的首次随访天数和儿童计划类型自动生成的儿童询问计划。满月的计划日期=出生日期+首次随访天数。

2)    管辖机构默认为儿童档案的管辖机构，不可修改。

3)    儿童询问信息中的询问日期的选择是根据系统配置管理->儿童模块参数设置界面中配置的提前天数和延后天数进行控制的，随访日期可选择的最小值是当前计划的开始日期（开始日期=当前计划日期-提前天数），随访日期最大值选择的条件是当前计划的结束日期（结束日期=当前计划日期+延后天数）和当前日期的比较，若结束日期＞当前日期，则最大值只能选择当前日期，若结束日期＜当前日期，最大值选择结束日期。

4)    实足日龄=当前计划日期-出生日期。
