6.4.1.4 家庭档案迁移申请

1.      在档案迁移管理主界面，申请人点击“申请”按钮，打开申请窗口，选择档案类别为“家庭档案”，如下图：

2.      当申请家庭档案迁移时，需要选择迁移的户主信息，在申请界面点击户主选项的选择按钮，弹出家庭信息查询界面，如下图：

3.      根据各种查询条件查询过滤本系统中所有的家庭档案，勾选需要迁移的家庭记录确定后，回到迁移申请界面，如下图：

1)     档案编号、户主、原网格地址、原责任医生和原管辖机构根据所选的家庭自动默认。

2)     当原管辖机构或现管辖机构与当前申请人所属管辖机构都不一样时，不允许申请。

3)     迁移申请保存后，申请单的迁移状态默认为待迁移，待迁移的申请单允许修改。

4)     中心内的迁移申请保存后，需要由本中心的防保科长进行确认操作。中心外的迁移申请保存后，由被申请方中心的防保科长进行确认。

注意：当系统管理-管理方式设置中网格化管理方式为“全部网格化管理”时，才能对家庭档案进行迁移，如果网格化管理方式为“部分网格化管理”时，不能对家庭档案迁移。
