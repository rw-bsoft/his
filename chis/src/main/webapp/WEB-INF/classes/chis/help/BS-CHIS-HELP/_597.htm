11.2.2.3 人员注册

在人员注册界面左侧选择机构，点击新建按钮，可打开人员新建界面，如下图：

1)     人员编号只能由数字、字母、_、- 组成；人员编号在系统中不允许重复；人员信息保存后人员编号不允许再修改。

2)     右键人员照片可以选择本地图片上传，人员照片允许上传gif、jpg、jpeg、bmp、png类型的最大不超过10M的文件。

3)     同一个身份证号码的人不允许重复进行人员注册。

4)     所在机构默认为新建时选择的机构名称，可以进行修改。

5)     拼音助记、五笔码和角型码根据人员姓名自动生成，不允许修改。

6)     有用户关联的人员不允许注销。
