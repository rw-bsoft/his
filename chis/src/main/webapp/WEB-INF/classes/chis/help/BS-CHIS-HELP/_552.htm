10.1.1.2 管理权限

体检管理模块，只有责任医生、团队长、儿保医生、妇保医生、中心主任、网络管理员、防保科长和市区卫生局角色可以访问。其中：

1)     责任医生、团队长用户登录：体检列表数据按管辖机构（中心）过滤；个档所属中心内的责任医生可以新建、修改和作废本中心内的体检记录。

2)     儿保医生用户登录：体检列表数据按管辖机构（中心）过滤；个档所属中心内的儿保医生可以新建、修改和作废本中心内的体检记录。

3)     妇保医生用户登录：体检列表数据按管辖机构（中心）过滤；个档所属中心内的儿保医生可以新建、修改和作废本中心内的体检记录。

4)     中心主任、网络管理员、防保科长登录：体检列表数据按管辖机构（中心）过滤；对体检记录只读。

5)     市区卫生局登录：体检列表数据按管辖机构（市/区）过滤；对体检记录只读。
