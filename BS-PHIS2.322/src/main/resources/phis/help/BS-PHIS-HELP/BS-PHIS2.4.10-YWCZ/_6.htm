1.5 推荐配置

l  应用服务器操作系统：Windows Server 2003、Windows Server 2012

l  数据库服务器操作系统：Windows Server 2008

l  应用软件：JDK1.7、Zookeeper3.4.3、metamorphosis-server-wrapper、Tomcat6.0、WebLogic10.3.3

l  数据库软件：Oracle11g

l  浏览器：Mozilla Firefox29.0及以上版本、Internet Explorer8.0及以上版本

l  屏幕分辨率：1366*768、1024*768
