9.10.2.3 操作步骤

1.  统计

    单击日期图标或者手动修改开方日期，单击【统计】，主界面显示该日期段出院病人在住院期间抗菌药物使用情况。

2.  打印

单击【打印】，成功打印主界面显示的住院抗菌药物使用数据。

3.  抗菌药物使用率 = 抗菌药物使用人数/出院人数

一级使用率 = 一级使用人数/抗菌药物使用人数

二级使用率 = 二级使用人数/抗菌药物使用人数

三级使用率 = 三级使用人数/抗菌药物使用人数

平均使用品种数:每个病人平均使用品种数

抗菌药物占药比 = 抗菌药物总费用/药品总费用

抗菌药物人均费用 = 抗菌药物总费用/出院人数

住院人天数 = 出院病人住院天数之和
