9.4.3 操作步骤

1.    查询

1） 单击下拉框选择机构，可以按机构或团队显示。

2） 单击时间框修改统计时间。

3） 单击【查询】可显示符合条件的结果。

2.    查询当前机构下的医生情况

双击列表中的某一项机构，即可显示当前机构下的医生，如图9-4-2所示：

 

图9-4-2

单击右上角的后退按钮又可回到当前机构。

3.   指标数据来源

    1）电子健康档案建档率 =建立电子健康档案人数/辖区内常住居民数×100％。

       注：辖区内常住居民由指标数据维护录入,电子健康档案数不计算已经注销的人数。

    2）健康档案合格率=抽查填写合格的档案份数/抽查档案总份数×100％。

       注：抽查填写合格的档案份数、抽查档案总份数由指标数据维护录入。

3）健康档案使用率=抽查档案中有动态记录的档案份数/抽查档案总份数×100％。

       注：抽查档案中有动态记录的档案份数、抽查档案总份数由指标数据维护录入。

4）高血压管理率 =年内已管理高血压人数/年内辖区内高血压患者总人数×100％。

       注：辖区高血压患病总人数估算：辖区常住成年人口总数×成年人高血压患病率（通过当地

       流行病学调查、社区卫生诊断获得或是选用本省（区、市）或全国近期高血压患病率指标）。

5）高血压规范管理率 = 按照规范要求进行高血压患者管理的人数/年内管理高血压患者人数

       ×100％。

       注：规范管理要求：档案内容填写无漏项，随访按计划完成，且内容无遗漏（测量血压、血

       糖，用药指导，生活方式等非药物指导）

6）高血压发现率：辖区内发现的高血压人数/辖区服务人口数×100%；

       注：辖区内发现的高血压人数:指标数据维护录入或者高血压管理数（数据来源：报表指标数

       据高血压管理数）；辖区服务人口数：常住人口（卫统中第六次人口普查）；

7）高血压控制率 = 最近一次随访血压达标人数/已管理的高血压人数×100％。

8）糖尿病管理率 = 年内已管理糖尿病患者人数/年内辖区内糖尿病患者总人数×100％。

       注：辖区内糖尿病患者总人数估算：辖区常住成年人口总数×成年人糖尿病患病率（通过当

       地流行病学调查、社区卫生诊断获得或是选用本省（区、市）或全国近期2型糖尿病患病率

       指标）。

9）糖尿病规范管理率 = 按照要求进行糖尿病患者健康管理的人数/年内管理糖尿病患者人数

       ×100％。

10）糖尿病发现率 = 辖区内发现的糖尿病人数/辖区服务人口数×100%；

注：辖区内发现的糖尿病人数:指标数据维护录入或者糖尿病管理数（数据来源：报表指标数据糖尿病管理数）；辖区服务人口数：常住人口（卫统中第六次人口普查）；

11）糖尿病控制率 = 最近一次随访空腹血糖达标人数/已管理的糖尿病患者人数×100％。

12）精神病管理率 = 所有登记在册的确诊重性精神疾病患者数/（辖区内15岁及以上人口总数

    ×患病率）×100％。

13）精神病规范管理率 = 每年按照规范要求进行管理的确诊重性精神疾病患者数/所有登记在册

        的确诊重性精神疾病患者数×100％。

14）精神病发现率 = 辖区内发现的精神病人数/辖区服务人口数×100%；

注：辖区内发现的精神病人数:指标数据维护录入或者精神病管理数（数据来源：报表指标

数据精神病管理数）；辖区服务人口数：常住人口（卫统中第六次人口普查）；

15）精神病稳定率 = 最近一次随访时分类为病情稳定的患者数/所有登记在册的确诊重性精神疾

        病患者数×100％。

16）精神病治疗率 = 最近一次随访时治疗（含规律和间断服药）的患者数/所有登记在册的确诊

        重性精神疾病患者数×100％；

注：最近一次随访时治疗（含规律和间断服药）的患者数：系统随访中最近一次随访时治疗（含规律和间断服药）的精神病档案数（手工录入或系统对接）；所有登记在册的确诊重性精神疾病患者数：系统中精神病档案管理数或者由指标数据维护录入；

17）老年人管理率 = 接受健康管理的60岁及以上常住老年人口数/年内辖区内60岁及以上常住

        老年人口总数×100%；

注：接受健康管理的60岁及以上常住老年人口数：当前机构下老年人健康档案数（数据来源：报表指标数据老年人档案管理数）；

18）老年人规范管理率 = 每年按照规范要求进行管理老年人人数/系统管理的老年人人数×100％。
