10.1.3 操作步骤

1.  选择需要维护的指标

2.  【新建】模板，如图10-1-2所示，录入对应的指标数据

   

                                          图10-1-2

3.  已经新建成功的指标数据模板，可以【修改】或【删除】

注：【修改】或【删除】的数据，只会对以后的采集有效，之前已经采集的数据不会改变。

4.  数据采集运行前，需要将“社区管理指标、孕产妇保健和健康情况年报表、七岁以下儿童保健和健康年报表、妇幼管理指标”的指标数据全部维护好，并包含所有对应下属机构或团队。
