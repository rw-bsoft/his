7.3.2 主界面

进入“统计分析|全院查询|全院收入情况”模块，主界面如图7-3-1所示：

 

图7-3-1

 1）统计的时间类别分三种：收费日期(SFRQ)、结帐日期(JZRQ)和汇总日期(HZRQ),可以在系统参

    数设置该值，参数名ETLRQLX(公有参数）

 2）医疗收入、药品收入和其他收入按收费项目的大类进行区分。

 3）如图，主界面显示本机构各科室的收入情况列表，列表上方为查询条件及功能按钮。

注：门诊收费和住院记账的数据都要统计，门诊收费需要用到上述参数，住院记账只采用汇总日期。
