10.2.3 操作步骤

1.  选择查询字段，可选择使用参数名称搜索

2.  参数名称：MZDCFJE

   门诊大处方金额（公有参数）：该参数用于判断首页门诊大处方金额指标的金额界限。

3.  参数名称：ETLRQLX

   统计分析时间口径（公有参数）：该参数主要用于ETL采集数据的时间条件类别，提供收费日期(SFRQ)、结帐日期(JZRQ)和汇总日期(HZRQ)三个选项。涉及模块：首页-医疗总收入、全院收入情况、门诊费用分析、门诊医生工作量统计、疾病统计分析、出院病人费用分析、住院性质费用汇总表、出院病人汇总表。

   注：首页-医疗总收入、全院收入情况，里面住院金额相关的采集只使用汇总日期。

4.  参数名称：GHCSFYGB

   挂号产生费用归并（公有参数）：就诊费、专家费、病历费等由挂号产生的费用通过参数设置制定到归并项目。

5.  参数名称：ETLBEGINDATE

   统计分析数据传送的开始时间（公有参数）：该参数主要用于ETL按时间段方式采集。

6.  参数名称：ETLENDDATE

   统计分析数据传送的结束时间（公有参数）：该参数主要用于ETL按时间段方式采集。
