7.1.3 操作步骤

1. 查询全部/单个病人的门诊信息

1）  选择门诊类型，查询条件都置空，单击【查询】，查询到所有病人的门诊信息。

2）  选择门诊类型，按需要填写查询条件，单击【查询】，查询到相应条件的病人门诊信息。

2. 查询全部/单个病人的住院信息

1）  选择住院类型，查询条件都置空，单击【查询】，查询到所有病人的住院信息，如图7-1-2所

    示：

        

图7-1-2

 

    2)   选择住院类型，按需要填写查询条件，单击【查询】，查询到相应条件的病人住院信息。

3.  查看门诊/住院的病人电子档案

    1）  选择门诊列表内的某条数据双击或者单击【EHRView】打开电子档案浏览器，可查看到病人

的基本信息、挂号记录、处方、检查单、医技结果报告、门诊病历等信息，如图7-1-3所示：

 

图7-1-3

 2）  选择住院列表内的某条数据双击或者单击【EHRView】打开电子档案浏览器，可查看到病人

 的基本信息、住院费用、电子病历等信息，如图7-1-4所示：

 

图7-1-4

4.  重置

    单击【重置】，将查询条件置空且默认选择门诊。
