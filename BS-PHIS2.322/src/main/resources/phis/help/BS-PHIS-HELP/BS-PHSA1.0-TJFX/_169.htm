9.1.3 操作步骤

1.   查询

单击下拉框选择一个或多个查询条件，并单击【查询】列表中显示符合条件的结果。

2.   指标数据来源

1）活产数：该指标取至系统数据维护模块，数据维护维度为中心机构和年度，分为男、女、性

   别不明活产数三个指标来维护。

2）产妇数：该指标取至系统数据维护模块，数据维护维度为中心机构和年度，分为农业户籍和

   非农业户籍两个指标来维护。

3）孕产妇保健管理情况

      （1）产妇建卡人数：取至孕产妇管理中有产后访视记录的孕产妇档案数。

      （2）产妇产前检查人数：该指标取至系统孕产妇产检随访中接受过1次及以上产前检查且有

       产后访视记录的孕产妇档案数。

      （3）产妇产前检查5次及以上人数：该指标取至系统孕产妇产检随访中接受过5次及以上产

       前检查且有产后访视记录的孕产妇档案数。

      （4）产妇孕早期产前检查人数：该指标取至系统孕妇档案中建册孕周小于13周且接受过一次

       及以上产前检查同时有产后访视记录的孕产妇档案数。

      （5）产妇孕产期中重度贫血人数：该指标取至孕产妇产前检查随访中血红蛋白含量小于90克

       /升且有产后访视记录的孕产妇档案数。

      （6）产妇艾滋病病毒检测人数：该指标取至接受过孕产妇产前检查随访中孕妇检查（HIV抗

       体检测）且有产后访视记录的孕产妇档案数。

  （7）孕产妇艾滋病病毒感染人数：孕产妇艾滋病病毒感染人数包括孕期至产时艾滋病病毒抗

   体确证试验阳性的产妇人数,以及孕期艾滋病病毒抗体确证试验阳性的在孕28周前终止妊娠

   或失访的孕妇人数。该指标取至接受过孕产妇产前检查随访中孕妇检查（HIV抗体检测为阳

   性）的孕产妇档案数。

      （8）产妇梅毒检测人数：该指标取至接受过孕产妇产前检查随访中孕妇检查（梅毒血清学试

       验检测）且有产后访视记录的孕产妇档案数。

  （9）产妇梅毒感染人数：该指标取至接受过孕产妇产前检查随访中孕妇检查（梅毒血清学试

   验检测为阳性）且有产后访视记录的孕产妇档案数。

  （10）产妇乙肝表面抗原检测人数：该指标取至接受过孕产妇产前检查随访中孕妇检查（乙肝

   表面抗原检测）且有产后访视记录的孕产妇档案数。

  （11）产妇乙肝表面抗原阳性人数：该指标取至接受过孕产妇产前检查随访中孕妇检查（乙肝

   表面抗原检测为阳性）且有产后访视记录的孕产妇档案数。

      （12）孕产妇产前筛查人数：该指标取至系统数据维护模块，数据维护维度为中心机构和年度。

      （13）孕产妇产前筛查高危人数：该指标取至系统数据维护模块，数据维护维度为中心机构和

       年度。

      （14）孕产妇产前诊断人数：该指标取至系统数据维护模块，数据维护维度为中心机构和年度。

  （15）孕产妇产前诊断确诊人数：该指标取至系统数据维护模块，数据维护维度为中心机构和

   年度。

  （16）产妇产后访视人数：统计接受过一次及以上产后方式的孕产妇档案数。

  （17）产妇系统管理人数：系统中有五次级以上产检随访、一次及以上产后访视的孕产妇档案数。

4）接生情况

  （1）住院分娩活产数：该指标取至系统数据维护模块，数据维护维度为中心机构和年度。

  （2）剖宫产活产数：该指标取至系统数据维护模块，数据维护维度为中心机构和年度。

  （3）非住院分娩中新法接生活产数：该指标取至系统数据维护模块，数据维护维度为中心机

   构和年度。

      （4）新法接生活产数：该指标取至系统数据维护模块，数据维护维度为中心机构和年度。

5）孕产妇高危管理

      （1）高危产妇人数：该指标取至在整个孕产妇档案管理中有高危因素且有产后访视记录的孕

       产妇档案数。

      （2）高危产妇管理人数：该指标取至在整个孕产妇档案管理中有高危因素且有五次级以上产

       检随访、一次及以上产后访视的孕产妇档案数。

      （3）高危产妇住院分娩人数：该指标取至系统数据维护模块，数据维护维度为中心机构和年度。

6）孕产妇死亡

  （1）孕产妇死亡人数：统计系统中孕产妇死亡报告卡卡片数。

7）围产儿情况

  （1）低出生体重儿数：该指标取至系统数据维护模块，数据维护维度为中心机构和年度。

  （2）死胎数：该指标取至系统数据维护模块，数据维护维度为中心机构和年度。

  （3） 死产数：该指标取至系统数据维护模块，数据维护维度为中心机构和年度。

  （4） 早期新生儿死亡数：该指标取至系统数据维护模块，数据维护维度为中心机构和年度。

  （5）围产儿死亡数：该指标取至系统数据维护模块，数据维护维度为中心机构和年度。

（孕产妇产前筛查率仅包括血清学筛查，不包括超声学筛查）

      （23） 高危产妇管理百分比=×100%

      （24） 高危产妇住院分娩百分比=×100%

      （27） 低出生体重儿百分比=×100%
