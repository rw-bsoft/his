6.1.2 主界面

在首页，查看医疗总收入的同时，也可以直接点击金额追溯查看具体科室明细，主界面如图6-1-2所示，列表上方为查询条件及功能按钮：

 

 

                                     图6-1-2

1) 统计的时间类别分三种：收费日期(SFRQ)、结帐日期(JZRQ)和汇总日期(HZRQ),可以在系统参数设置该值，参数名ETLRQLX(公有参数）

注：门诊收费和住院记账的数据都会统计，门诊收费需要用到上述参数，住院记账只采用汇总日期。

2) 可以设置参数GHCSFYGB : 病历费、专家费等归并（1：医疗，2：药品，3：其他，0：挂号费），就诊费、专家费、病历费等由挂号产生的费用通过参数设置制定到归并项目；
